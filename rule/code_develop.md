# 代码开发规范

### 边界值

```bash
1. 列表要考虑列表为空或超出一页
2. 文字要考虑文字为空或超出一行
3. 对象要考虑是否为 undefined
```

### 特殊场景

```bash
1. 交互状态：鼠标移入，点击状态，是否禁用，是否需要文字提示
2. 浮点数：计算精度
3. 防抖节流：防止重复提交，触发
4. 最低代价：无法避免负面影响时，选择代价最小的那个
```

### 魔法值问题

```javascript
// 魔法值：程序中突然出现的值
if(res.code === 200){}

// 枚举代替魔法值
enum Code {
  SUCCESS = 200,
  ERROR = 500
}
if(res.code === Code.SUCCESS){}
```
